# 이 프로젝트에 대해

## 개요
개발 공부용 연습 폴더입니다.

- **가계부** (`index.html`) — 바닐라 HTML/JS로 만든 가계부 앱
- **고구마마켓** (`goguma-market/`) — Next.js로 만드는 중고거래 웹앱 (당근마켓 스타일)

## 기술 스택
- **Next.js** — 웹사이트를 만드는 React 프레임워크
- **Supabase** — 로그인·데이터베이스를 쉽게 쓸 수 있는 서비스 (프로젝트 ID: `fqtmbthlwmmmcihgdnhl`)
- **Vercel** — 완성된 웹사이트를 인터넷에 올리는 배포 서비스
- **Tailwind CSS** — 클래스 이름으로 디자인하는 CSS 도구

# 프로젝트 현황

## 고구마마켓 개발 현황
| 단계 | 내용 | 상태 |
|------|------|------|
| 1단계 | 회원가입 / 로그인 / 로그아웃 | ✅ 완료 |
| 2단계 | 상품 등록 / 목록 보기 | 🔜 예정 |
| 3단계 | 상품 상세 / 수정 / 삭제 | 🔜 예정 |
| 4단계 | 채팅 | 🔜 예정 |
| 5단계 | Vercel 배포 | 🔜 예정 |

## 주요 파일 위치
```
ggm/
├── index.html                  ← 가계부 앱
└── goguma-market/              ← 고구마마켓 Next.js 앱
    ├── proxy.ts                ← 로그인 여부 확인 (페이지 보호)
    ├── lib/supabase/           ← Supabase 연결 설정
    ├── app/actions/auth.ts     ← 회원가입·로그인·로그아웃 기능
    ├── app/(auth)/             ← 로그인·회원가입 화면
    └── app/(main)/             ← 로그인 후 화면
```

## 개발 서버 실행
```bash
cd goguma-market
npm run dev
# → http://localhost:3001 에서 확인
```

## ⚠️ 사람이 직접 해야 하는 설정
**이메일 인증 끄기** (개발 중에는 불편하므로 꺼두세요):
1. [Supabase 대시보드](https://supabase.com/dashboard) 접속
2. 프로젝트 선택 → 왼쪽 메뉴 **Authentication** 클릭
3. **Providers** → **Email** 클릭
4. **"Confirm email"** 스위치를 **OFF**로 변경
5. Save 클릭
